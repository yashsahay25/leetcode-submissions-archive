# leetcode-submissions-archive

Historical archive of my accepted LeetCode submissions.

This repository is the storage hub for solution files and timestamped Git history. It is intended to be populated by the companion sync tool:

```text
yashsahay25/leetcode-sync-tool
```

## Purpose

This repository stores accepted LeetCode submissions as source files. Each archived submission is expected to include the LeetCode submission ID in its filename so multiple accepted solutions for the same problem can coexist without overwriting each other.

Example:

```text
1_two-sum_123456789.cpp
1_two-sum_987654321.cpp
```

## Sync Source

Submissions are fetched from LeetCode and committed here by `leetcode-sync-tool`, which preserves original LeetCode submission timestamps in the Git commit history.
